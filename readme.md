### Requirements
- Node
- dropbox-app Authorization token: generate from their [app console](https://www.dropbox.com/developers/apps?_tk=pilot_lp&_ad=topbar4&_camp=myapps)

### Install 
`npm i -g jvb-cli-dbx` or by yarn / your package manager.

### Usage:
```sh
dbx connect
```
```sh
dbx token:set
dbx token:destroy
```
